LifeManager UI モック — 直感操作リデザイン案

既存のダークパレット（#0F172A / #1E293B ＋ Teal / Violet）を活かし、「化粧パレットのように見た瞬間に何をどこで操作するか分かる」ことを狙った提案です。収入＝ティール、支出＝ローズで役割を色分けし、合計は符号で色が変わります。

配色の役割

収入・プラス（ティール） 支出・マイナス（ローズ） 主要アクション（バイオレット） 面・カード（スレート）

直感操作のための設計方針

  • 色で役割が即わかる：ラベルを読まなくても収入／支出／保存を判別できる。
  • 合計が符号と色で反応：プラスはティール、マイナスはローズに切り替わる。
  • 操作の縦導線を1本化：日付→収入→支出→合計→保存を上から順に。
  • ホームは大きな2択タイル：アイコン＋短い説明で迷いを減らす。

※ これはデザイン確認用の静的モックです。実装（XAML への反映）は方向性合意後に別対応します。
